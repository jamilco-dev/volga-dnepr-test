Файл *.dat - сама полетная информация,
*.cyc - циклограмма.

Основное из циклограммы:
Кадр (соответствует 1 сек времени) - это последовательность из 256 слов (по 2 байта),
каждое слово это канал передаваемого параметра. Параметр HGRV - Высота, измеряемая радиовысотомером. 
Значения передаются по двум каналам 6 и 134, маска 0FFFh. 
Физическое значение получается путем умножения значения кода на 0,3048.

Предлагаю найти диапазон кадров (продолжительность времени)  когда воздушное судно находилось на высоте более 10м.

Для упрощения, думаю, можно использовать только один канал высоты.
